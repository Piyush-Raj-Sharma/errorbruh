# ErrorBruh 😱 — Meme Sounds for Code Errors

Turn your coding pain into comedy.

**ErrorBruh** plays hilarious meme sounds whenever your code fails, throws errors, or disrespects the compiler. Experience emotional damage in surround sound.

---

## 😂 Features

- 🔊 Plays meme sounds on terminal command failure  
- 🧠 Plays sounds on syntax / compile errors  
- 🎛 Fully customizable sound selection  
- ⏱ Built-in cooldown to prevent spam  
- ⚡ Lightweight and fast  
- 😈 Emotionally damaging developer experience  

---

## 🔥 Available Sounds

- **FAAAH 😱** — Viral emotional damage scream  
- **Hey Maa Mataji 🙏** — Divine reaction to broken code  
- **Jetha Nonsense 🤦** — Compiler disappointed in you  
- **Padhaai likhaai me dhyan do 📚** — Skill issue detected  
- **Mat sikha chal 😤** — Compiler got offended  
- **Silent mode 🤫** — Suffer quietly  

---

## ⚙️ Extension Settings

Open:
Settings → Search → ErrorBruh


Configure:

- Terminal Error Sound
- Syntax Error Sound

Choose your preferred emotional damage level.

---

## 💻 How it works

ErrorBruh listens to:

- Terminal command execution failures
- VS Code diagnostics (syntax / compile errors)

When an error is detected, it plays your selected meme sound instantly.
